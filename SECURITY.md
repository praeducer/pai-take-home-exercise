# Security Policy

## Supported Scope

This repository is a public PoC. Security controls are applied to source control hygiene, secret handling, assistant governance, and AWS operational safeguards.

## Reporting a Vulnerability

If you discover a potential vulnerability, please report it privately to the maintainer through repository contact channels.

When reporting, include:

- A clear description of the issue.
- Steps to reproduce.
- Potential impact.
- Suggested remediation, if available.

Please do not disclose vulnerabilities publicly until they have been reviewed.

## Response Guidelines

- Reports are triaged as quickly as possible.
- Verified issues are prioritized by severity and exploitability.
- Fixes aim for least-privilege and defense-in-depth outcomes.
