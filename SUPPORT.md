# Support

## Getting Help

Use GitHub Issues for questions, bugs, and enhancement requests.

Please include:

- What you were trying to do.
- Relevant environment details.
- Any non-sensitive logs or error messages.

## Security Issues

For suspected vulnerabilities, follow the process in `SECURITY.md`.

## Project Scope

This repository is a PoC. Some production-grade features are intentionally deferred and tracked in `BACKLOG.md`.
